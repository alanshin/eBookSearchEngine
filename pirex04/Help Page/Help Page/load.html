How to Load Documents

  1. Click the "Browse Button"
  2. Locate desired file and select it
  3. Select the file type from the "Text File Type Dropdown"
  4. Author and Title will be automatically pulled from the document
  5. Click the "Process Button"
  6. A summary of the load will apppear in the "Load Summary Window"
